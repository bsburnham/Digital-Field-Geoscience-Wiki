## Getting Started

### Before You Begin

1. **Familiarise yourself with the wiki** - Browse the existing content to understand the structure and style
2. **Check existing issues** - See if someone is already working on what you'd like to contribute
3. **Join the discussion** - Use [GitHub Discussions](https://github.com/bsburnham/Digital-Field-Geoscience-Wiki/discussions) to propose major changes or new sections
4. **Read our [Code of Conduct](CODE_OF_CONDUCT.md)** - All contributors must follow our community standards

### How to Contribute

1. **Fork the repository** to your GitHub account
2. **Create a new branch** for your changes:
   ```bash
   git checkout -b add-your-feature
   ```
3. **Make your changes** following the content guidelines below
4. **Commit your changes** with a clear message:
   ```bash
   git commit -m "Add section on RTK GPS field methods"
   ```
5. **Push to your fork**:
   ```bash
   git push origin add-your-feature
   ```
6. **Open a Pull Request** with a description of your changes

### Adding Yourself as a Contributor

When you make a contribution, please add yourself to the [Authors](AUTHORS.md) list in your pull request. See the AUTHORS.md file for formatting guidelines.

## Code of Conduct

We're part of a professional geoscience community that conducts itself with mutual respect and consideration for others. All contributors must follow our [Code of Conduct](CODE_OF_CONDUCT.md), which aligns with the principles of the Geological Society of London and the broader geoscience community.
